## 🌟 About

**ScanCraft** is a modern QR & Barcode Generator built with **HTML, CSS, and JavaScript**.

Generate high-quality QR codes and barcodes with a clean UI, customize their appearance, preview them instantly, and download them in seconds.

This project is **Day 4** of my **"30 Days, Build Something New"** challenge.

---

## ✨ Features

- 🔳 Generate QR Codes
- 📦 Generate Barcodes
- 🎨 Customize foreground & background colors
- 📐 Multiple size options
- 👀 Live Preview
- 🌙 Dark Mode
- 📥 Download generated codes
- 🔄 Reset everything instantly
- 📱 Responsive Design
- ⚡ Fast & Lightweight
